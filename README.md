
#  C++ File Integrity Checker

A simple cybersecurity-focused C++ project that calculates SHA-256 hashes
to detect file modifications.

##  Build

```bash
mkdir build
cd build
cmake ..
make
```

##  Run

```bash
./file_integrity_checker <file>
```

##  What You Learn
- Secure file I/O
- Cryptographic hashing
- OpenSSL usage
- CMake basics

 Educational purpose only.
